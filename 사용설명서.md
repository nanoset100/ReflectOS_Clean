## 1. 앱 시작하기

### 🖥️ 앱 실행 방법

1. **터미널**(명령 프롬프트)을 열어요.
2. 프로젝트 폴더로 이동해요:
   ```
   cd M:\MyProject777\ReflectOS_Clean
   ```
3. 앱을 실행해요:
   ```
   streamlit run app.py
   ```
4. 브라우저가 자동으로 열리고 앱이 나타나요! 🎉

### 🌐 앱 접속 주소

- `http://localhost:8501`

### 🔐 처음 사용할 때

- **회원가입**: 이메일과 비밀번호(6자 이상)로 가입해요.
- **로그인**: 가입한 이메일/비밀번호로 로그인하면 홈 화면으로 이동해요.

## 3. Check-in - 오늘 기록하기

> ✍️ **Check-in**은 오늘의 생각과 기분을 기록하는 곳이에요!

### 📝 기록 방법 (순서대로)

#### Step 1: 내용 입력하기

텍스트 박스에 오늘 있었던 일이나 생각을 적어요.

**예시:**
```
오늘 수학 시험을 봤는데 생각보다 잘 본 것 같아서 기분이 좋다.
내일은 영어 단어 외워야겠다.
```

#### Step 2: 기분 선택하기

이모지를 클릭해서 오늘의 기분을 선택해요.

| 이모지 | 의미 |
|--------|------|
| 😄 | 아주 좋음 |
| 🙂 | 좋음 |
| 😐 | 보통 |
| 😔 | 안 좋음 |
| 😢 | 아주 안 좋음 |

#### Step 3: 에너지 레벨 선택하기

슬라이더를 움직여서 오늘의 에너지를 표시해요.

- **1** = 매우 피곤함 😴
- **5** = 보통
- **10** = 매우 활력 넘침 ⚡

#### Step 4: 태그 입력하기 (선택)

관련된 키워드를 입력해요. **예시:** `공부, 시험, 수학`

#### Step 5: 저장하기

**💾 저장하기** 버튼을 클릭하면 끝! 🎉

---

### 🎤 음성으로 기록하기 (멀티모달)

글 쓰기 귀찮을 때는 **음성**으로 기록할 수 있어요.

1. 🎙️ **음성 입력** 섹션에서 파일 업로드
2. 녹음 파일(mp3, wav)을 선택
3. AI가 자동으로 글로 변환해줘요!

### 🖼️ 이미지로 기록하기

손으로 쓴 메모나 사진도 기록할 수 있어요.

1. 🖼️ **이미지 입력** 섹션에서 파일 업로드
2. 이미지 파일(jpg, png)을 선택
3. AI가 이미지를 분석해서 내용을 추출해요!

## 4. Report - 주간 리포트

> 📊 **Report**는 일주일 동안의 기록을 AI가 분석해주는 곳이에요!

### 📋 사용 방법

#### Step 1: 기간 선택하기

- **이번 주** / **지난 주** / **지난 2주** 버튼으로 기간을 정하거나, 날짜를 직접 선택할 수 있어요.

#### Step 2: 리포트 생성하기

**📝 리포트 생성** 버튼을 클릭해요. AI가 분석하는 동안 잠시 기다리면 돼요.

#### Step 3: 결과 확인하기

| 섹션 | 설명 |
|------|------|
| 💬 **한 줄 요약** | 이번 주를 한 문장으로! |
| 🏆 **이번 주 성취** | 내가 잘한 것들 |
| ⚠️ **어려웠던 점** | 힘들었던 것들 |
| 🔄 **발견된 패턴** | 반복되는 습관이나 행동 |
| 🚀 **다음 주 제안** | AI의 조언 |

---

## 5. Planner - 시간 계획

> 📅 **Planner**는 AI가 하루 일정을 계획해주는 곳이에요!

### 📋 사용 방법

#### Step 1: 목표 입력하기

오늘 하고 싶은 일을 적어요.

**예시:**
- 목표 1: `영어 단어 50개 외우기`
- 목표 2: `운동 30분 하기`

#### Step 2: AI 계획 생성하기

**🤖 AI 계획 생성** 버튼을 클릭해요.

#### Step 3: 시간표 확인하기

AI가 만들어주는 시간 블록 예시:

| 시간 | 할 일 | 카테고리 |
|------|-------|----------|
| 09:00~10:00 | 아침 루틴 | 🏠 생활 |
| 10:00~12:00 | 영어 공부 | 💻 업무 |
| 12:00~13:00 | 점심 식사 | ☕ 휴식 |
| 15:00~15:30 | 운동 | 🏃 건강 |

#### Step 4: 계획 저장하기

마음에 들면 **💾 계획 저장** 버튼을 클릭!

---

### 💡 실행 팁

- AI가 목표를 달성하기 위한 **팁**도 함께 알려줘요.
- Google Calendar를 연결해두면, 이미 있는 일정을 피해서 블록을 제안해줄 수 있어요.

## 6. Memory - 기억 검색

> 🧠 **Memory**는 과거 기록을 AI가 찾아주는 곳이에요!

### 📋 사용 방법

#### Step 1: 궁금한 것 물어보기

질문을 입력해요.

**좋은 질문 예시:**
- "내가 자주 미루는 이유는?"
- "지난달 성취한 것들은?"
- "내가 기분 좋았던 날은 언제야?"
- "영어 공부 관련 기록 찾아줘"

#### Step 2: 검색하기

**🔎 검색하기** 버튼을 클릭해요.

#### Step 3: 답변 확인하기

AI가 과거 기록을 찾아서 답변해줘요!

### 🔄 체크인 동기화

처음 사용할 때는 **📥 체크인 동기화** 버튼을 한 번 눌러주세요!  
과거 체크인들이 검색용 인덱스에 반영돼요.

### ⚙️ 사이드바 설정

- **검색 결과 수**: 유사한 기억을 몇 개까지 볼지 (3~10)
- **유사도 임계값**: 이 값 이상인 것만 표시
- **🧪 데모 데이터 제외**: 테스트용 기록은 검색에서 제외

## 7. Settings - 설정

> ⚙️ **Settings**는 앱 설정과 연동을 관리하는 곳이에요!

### 🔗 연결 상태 확인

| 서비스 | 설명 |
|--------|------|
| 🗄️ **Supabase** | 데이터 저장소 |
| 🤖 **OpenAI** | AI 기능 |
| 📅 **Google Calendar** | 캘린더 연동 (선택) |

### 📅 Google Calendar 연결하기 (선택)

1. **🔑 Google 계정으로 연결** 버튼 클릭
2. Google 로그인 후 권한 허용
3. 연결 완료! Home·Planner에서 일정을 활용할 수 있어요.

### 🔌 모듈 활성화 (중요!)

**건강 / 수험생 / 취준생** 모듈은 여기서 켜야 메뉴에 나타나요.

1. **Settings** 화면에서 **🔌 모듈 활성화** 섹션으로 내려가요.
2. 사용할 모듈에 체크해요:
   - 🏃 **건강관리**: 식단·운동·체중 기록
   - 📚 **수험생**: 학습 기록·과목 목표·슬럼프 로그
   - 💼 **취준생**: 지원 현황·면접 기록·이력서 관리
3. **💾 모듈 설정 저장** 버튼을 누르고, 페이지를 새로고침하면 사이드 메뉴에 해당 모듈 메뉴가 생겨요.

## 8. 건강 모듈 사용하기

> 🏃 **건강 모듈**은 식단·운동·체중을 기록하고, 그래프와 리포트로 확인하는 기능이에요.  
> **Settings → 모듈 활성화**에서 "건강관리"를 켜면 **오늘 기록 / 체중 / 운동 / 건강 리포트** 메뉴가 나타나요.

---

### 8-1. 📝 오늘 기록

**메뉴:** 오늘 기록 (health 모듈)

식단·운동·체중을 **한 화면에서 탭**으로 나눠 입력해요.

#### 🍽️ 식단 탭

| 입력 항목 | 설명 |
|-----------|------|
| 날짜 | 기록할 날짜 |
| 식사 종류 | 아침 / 점심 / 저녁 / 간식 |
| 식단 내용 | 먹은 음식 (예: 밥, 된장국, 김치) |
| 칼로리 (kcal) | 선택 입력, 0이면 미입력 |

**💾 저장**을 누르면 해당 날짜의 식단이 기록돼요.

#### 🏋️ 운동 탭

| 입력 항목 | 설명 |
|-----------|------|
| 날짜 | 운동한 날짜 |
| 운동 종류 | 예: 러닝, 헬스, 수영, 요가 |
| 운동 시간 (분) | 실제 한 시간 |
| 운동 강도 | 낮음 / 보통 / 높음 / 매우 높음 |
| 메모 | 예: 5km 달리기 완료, 컨디션 좋음 |

**💾 저장**을 누르면 운동 로그가 기록돼요.

#### ⚖️ 체중 탭

| 입력 항목 | 설명 |
|-----------|------|
| 날짜 | 측정한 날짜 |
| 체중 (kg) | 숫자 입력 |
| 체지방률 (%) | 선택, 0이면 미기록 |
| 골격근량 (kg) | 선택, 0이면 미기록 |
| 메모 | 예: 아침 공복 측정 |

**💾 저장**을 누르면 체중 데이터가 기록돼요.

---

### 8-2. ⚖️ 체중

**메뉴:** 체중 (health 모듈)

**최근 30일** 체중 추이를 **선 그래프**로 볼 수 있어요.

- 그래프 아래에 **최근 체중 / 30일 변화 / 평균 체중**이 요약돼요.
- 데이터는 **오늘 기록 → 체중 탭**에서 입력한 내용이에요.

---

### 8-3. 🏋️ 운동

**메뉴:** 운동 (health 모듈)

**기간을 정해서** 운동 기록 목록을 볼 수 있어요.

- **시작일 / 종료일**로 기간을 선택해요.
- 목록에는 **운동 종류, 날짜, 시간(분), 강도, 메모**가 나와요.
- 하단 **요약**에서 **총 운동 시간**, **운동 종류 수**, **종류별 횟수**를 확인할 수 있어요.

---

### 8-4. 📈 건강 리포트

**메뉴:** 건강 리포트 (health 모듈)

**최근 7일** 건강 기록을 한눈에 요약해요.

| 섹션 | 내용 |
|------|------|
| ⚖️ 체중 변화 | 시작/현재 체중, 변화량 |
| 🏋️ 운동 통계 | 운동 횟수, 총/평균 시간, 종류별 횟수 |
| 🍽️ 식단 통계 | 식사 기록 횟수, 총/평균 칼로리, 종류별 횟수 |
| 💡 종합 평가 | 체중·운동·식단 기록 여부에 따른 한줄 코멘트 |

## 9. 수험생 모듈 사용하기

> 📚 **수험생 모듈**은 학습 세션·과목 목표·슬럼프를 기록하고, 학습 리포트로 통계를 보는 기능이에요.  
> **Settings → 모듈 활성화**에서 "수험생"을 켜면 **오늘 학습 / 과목 목표 / 슬럼프 로그 / 학습 리포트** 메뉴가 나타나요.

---

### 9-1. 📖 오늘 학습

**메뉴:** 오늘 학습 (student 모듈)

**학습 세션**을 하나씩 기록해요.

| 입력 항목 | 설명 |
|-----------|------|
| 날짜 | 공부한 날짜 |
| 과목 | 예: 수학, 영어, 국어 |
| 학습 시간 (분) | 그날 그 세션 공부한 시간 |
| 집중도 (1~5) | 1=매우 낮음, 5=매우 높음 |
| 학습 주제 | 쉼표로 구분 (예: 함수, 미적분, 삼각함수) |
| 메모 | 오늘 배운 점, 어려웠던 부분, 다음에 할 일 |

**💾 저장** 후 **최근 학습 세션** 목록에서 방금 저장한 세션을 포함해 최근 5개를 볼 수 있어요.

---

### 9-2. 📋 과목 목표

**메뉴:** 과목 목표 (student 모듈)

과목마다 **주간 목표 시간·우선순위·시험일**을 설정해요.

| 입력 항목 | 설명 |
|-----------|------|
| 과목 | 예: 수학, 영어 |
| 주간 목표 시간 (분) | 한 주에 공부하려는 시간 |
| 우선순위 | 높음 / 보통 / 낮음 |
| 시험일 | 선택, 있으면 입력 |
| 메모 | 과목별 특이사항, 학습 전략 등 |

저장하면 **과목 목표 목록**에 최신순으로 쌓여요. 우선순위는 🔴높음 / 🟡보통 / 🟢낮음으로 표시돼요.

---

### 9-3. 😔 슬럼프 로그

**메뉴:** 슬럼프 로그 (student 모듈)

공부가 잘 안 되거나 기분이 가라앉았을 때 **원인·증상·대처**를 기록해요.

| 입력 항목 | 설명 |
|-----------|------|
| 날짜 | 슬럼프를 느낀 날짜 |
| 기분 | 매우 나쁨 ~ 매우 좋음 |
| 슬럼프 원인/계기 | 예: 시험 결과 나쁨, 특정 과목 어려움 |
| 증상/느낌 | 예: 집중 안 됨, 불안감 |
| 시도한 대처 방법 | 예: 휴식, 운동, 친구와 대화 |
| 결과 | 개선됨 / 조금 나아짐 / 변화 없음 / 더 나빠짐 |
| 추가 메모 | 느낀 점, 다음에 시도할 것 |

저장 후 **최근 슬럼프 로그**에서 패턴을 되짚어 보세요.

---

### 9-4. 📊 학습 리포트

**메뉴:** 학습 리포트 (student 모듈)

## 10. 취준생 모듈 사용하기

> 💼 **취준생 모듈**은 지원 현황·면접 기록·이력서를 관리하고, 취준 리포트로 요약보는 기능이에요.  
> **Settings → 모듈 활성화**에서 "취준생"을 켜면 **지원 현황 / 면접 기록 / 이력서 관리 / 취준 리포트** 메뉴가 나타나요.

---

### 10-1. 📮 지원 현황

**메뉴:** 지원 현황 (jobseeker 모듈)

지원한 **회사·직무·상태**를 하나씩 기록해요.

| 입력 항목 | 설명 |
|-----------|------|
| 회사명 | 예: 네이버, 카카오 |
| 직무/포지션 | 예: 백엔드 개발자 |
| 지원 상태 | 지원 완료 / 서류 통과 / 면접 진행 / 최종 합격 / 불합격 / 포기 |
| 지원일 | 지원한 날짜 |
| 지원 링크 | 선택, 공고나 지원 페이지 URL |
| 메모 | 특이사항, 준비 내용, 후기 등 |

**지원 현황 목록**에서 **상태 필터**(전체/지원 완료/서류 통과 등)로 필터링할 수 있고,  
상단에 **전체 지원 / 서류 통과 / 면접 진행 / 최종 합격** 건수가 요약돼요.

---

### 10-2. 💬 면접 기록

**메뉴:** 면접 기록 (jobseeker 모듈)

면접 **날짜·차수·질문·자기 평가·다음 액션**을 기록해요.

| 입력 항목 | 설명 |
|-----------|------|
| 회사명 | 면접 본 회사 |
| 면접 차수 | 1차 / 2차 / 3차 / 최종 / 기타 |
| 면접일 | 날짜 |
| 면접 시간 | 선택, 시간까지 있으면 입력 |
| 면접 질문 | 질문을 한 줄에 하나씩 입력 |
| 자기 평가 | 매우 나쁨 ~ 매우 좋음 |
| 다음 액션 | 면접 후 해야 할 일, 개선할 점 |
| 추가 메모 | 특이사항, 느낀 점 |

저장하면 **최근 면접 기록** 목록에 쌓이고, **면접 질문 / 다음 액션 / 메모**는 펼쳐서 볼 수 있어요.

---

### 10-3. 📄 이력서 관리

**메뉴:** 이력서 관리 (jobseeker 모듈)

이력서 **버전별로 제목·작성일·내용 요약·메모**를 저장해요.

| 입력 항목 | 설명 |
|-----------|------|
| 이력서 제목 | 예: 백엔드 개발자 이력서 |
| 버전 | 예: v1.0, 2024-01 (선택) |
| 작성일 | 해당 버전 작성일 |
| 이력서 내용 요약 | 변경 사항, 강조 포인트 등 요약 |
| 메모 | 피드백, 개선 계획 등 |

**이력서 목록**은 최신순으로 보이고, **내용 요약 / 메모**는 펼쳐서 확인할 수 있어요.

---

### 10-4. 📊 취준 리포트

**메뉴:** 취준 리포트 (jobseeker 모듈)

**최근 7일** 지원·면접 기록을 요약해요.

| 섹션 | 내용 |
|------|------|
| 📮 지원 현황 | 전체/지원 완료/서류 통과/면접 진행/최종 합격 건수, 상태별 막대 그래프 |
| 💬 면접 통계 | 면접 횟수, 면접 본 회사 수, 회사별 면접 횟수 |
| 📌 다음 액션 Top 3 | 면접 기록에 적은 "다음 액션" 중 상위 3개 |
| 💡 종합 평가 | 지원·서류 통과·면접 기록 여부에 따른 한줄 코멘트 |

## 11. 꿀팁 & FAQ

### 🌟 꿀팁 모음

1. **매일 짧게라도 기록하기**  
   하루 5분만 투자해서 Check-in이나 모듈 기록을 해두면, 나중에 Memory·Report에서 많이 활용돼요.

2. **솔직하게 쓰기**  
   좋은 것만 쓸 필요 없어요. 힘들었던 것도 적으면 AI가 더 도움이 돼요.

3. **주간 리포트 확인하기**  
   Report(공통)와 모듈별 리포트(건강/학습/취준)를 주기적으로 보면 패턴을 파악하기 좋아요.

4. **목표는 구체적으로**  
   Planner에서 "공부하기"보다 "영어 단어 30개 외우기"처럼 구체적으로 쓰면 AI가 더 실천 가능한 계획을 세워줘요.

5. **모듈은 필요할 때만 켜기**  
   Settings에서 쓰는 모듈만 켜두면 메뉴가 단순해져서 쓰기 편해요.

### ❓ 자주 묻는 질문

**Q: 기록이 저장 안 돼요!**  
**A:** Home에서 "Supabase 연결됨"이 초록색인지 확인해보세요. Settings의 DB 상태 확인도 해보세요.

**Q: AI 기능이 작동 안 해요!**  
**A:** Settings에서 "OpenAI API 키 설정됨"이 초록색인지 확인해보세요.

**Q: 건강/수험생/취준생 메뉴가 안 보여요!**  
**A:** Settings → **🔌 모듈 활성화**에서 해당 모듈을 체크한 뒤 **💾 모듈 설정 저장**을 누르고, 페이지를 새로고침해보세요.

**Q: 앱이 안 열려요!**  
**A:** 터미널에서 `streamlit run app.py`를 다시 실행해보세요.  
중단하려면 터미널에서 `Ctrl + C`로 서버를 끈 다음 다시 `streamlit run app.py`로 실행하면 돼요.

---

### 📞 도움이 필요하면?

- 터미널에 나오는 **오류 메시지**를 확인해보세요.
- 대부분 **서버 재시작**으로 해결돼요:
  1. 터미널에서 `Ctrl + C`로 서버 중지
  2. `streamlit run app.py`로 다시 시작
